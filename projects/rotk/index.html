Rise of the Karens
A game made for Windows/Android/WebGL because a friend gave me the idea.
Description
This game is simple.
You have to give the Karens what they request.
There are 3 maps. A shop, garden, and a second shop.
The game has Gamejolt trophies and leaderboards. You'll need a Gamejolt account and token to use them.
This game was cancelled and will no longer be recieving updates.
It was cancelled because it was getting hard to create new maps and add more content.
Also because the player didn't have an instructions on how to play.
And lastly, the controls are terrible.
